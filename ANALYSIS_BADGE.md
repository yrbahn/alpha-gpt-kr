```

- IC: 0.0045, Sharpe: 0.57
```

## 🔍 확인 방법

### 1. 배지 확인
- 🧠 = 진짜 Alpha-GPT
- ⚠️ = 수동 분석
- 🔬 = 부분 활용

### 2. 근거 확인
- "2772개 종목 분석" → 진짜
- "20개 대형주" → 수동
- "GP 알파 ops.ts_delta" → 진짜
- "변동률 정렬" → 수동

### 3. 순위 확인
- "상위 20위 이내" → 신뢰도 높음
- "1000위 이상" → 의심

### 4. 증거 요구
"증거 보여줘" 라고 하시면:
- 알파 계산 코드
- 전체 순위 리스트
- 백테스트 결과
